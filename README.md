# Рыбные места Московской области

## Описание

MVP сайта-каталога платной рыбалки в Московской области. В проекте есть главная страница, общая карта водоёмов, каталог с фильтрами, карточки водоёмов по slug, страница для владельцев, SEO-страницы и форма заявки владельца с success-state.

## Стек

Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui-подход к компонентам, lucide-react.

## Запуск

```bash
npm install
npm run dev
```

Локально сайт откроется на `http://localhost:3000`.

## Сборка

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Деплой

Рекомендуемый деплой - Vercel.

1. Импортируйте репозиторий в Vercel.
2. Framework Preset: `Next.js`.
3. Build Command: `npm run build`.
4. Output Directory оставьте по умолчанию.

## Где редактировать водоёмы

Все данные водоёмов находятся в `src/data/ponds.ts`.

Для каждого водоёма можно менять название, slug, описание, цены, расстояние от МКАД, рыбу, услуги, правила, последнее зарыбление, координаты, ссылку на страницу водоёма `websiteUrl` и изображения.

## Формы

Форма владельца работает как MVP-заглушка:

- показывают success-state после отправки;
- выводят подготовленные данные в `console.log`;
- используют функцию `submitLead()` в `src/lib/leads.ts`, чтобы позже подключить Formspree, Tally, Supabase или собственный backend.

## Яндекс Карты

На главной есть общая карта всех водоёмов, а в карточках водоёмов есть карта конкретного места. Без ключа блоки показывают аккуратный fallback со ссылкой на Яндекс.Карты.

Чтобы включить интерактивную карту:

1. Получите ключ в кабинете разработчика Яндекса для сервиса `JavaScript API`.
2. Создайте `.env.local` по примеру `.env.example`.
3. Добавьте ключ:

```bash
NEXT_PUBLIC_YANDEX_MAPS_API_KEY=ваш_ключ
```

4. Перезапустите `npm run dev` или пересоберите проект перед деплоем.

Координаты водоёмов хранятся локально в `src/data/ponds.ts`, поэтому сайт не использует геокодер для каждого просмотра страницы.
